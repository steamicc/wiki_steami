# Ecran et audio

## Ecran OLED

| Composant | Description | Fabricant | Reference | Qte |
|-----------|-------------|-----------|-----------|-----|
| WEO128128BWPP3N00001 | Module OLED graphique blanc 128x128, COG (Chip on Glass), Longlife | Winstar | WEO128128BWPP3N00001 | 1 |

### Caracteristiques

- **Technologie** : OLED (pixels blancs)
- **Resolution** : 128 x 128 pixels
- **Montage** : COG (Chip on Glass)
- **Connecteur** : FFC/FPC 24 positions, pas 0.50 mm
- **Interface** : SPI1

| Composant | Description | Fabricant | Reference | Qte |

| SMT-0825-S-HT-R | Transducteur piezo montage en surface, 3.6V pic-a-pic | PUI Audio Inc | SMT-0825-S-HT-R | 1 |

### Caracteristiques

- **Type** : Transducteur piezo SMD

- **Alimentation** : Via convertisseur boost LMR62421 (14.5V)
- **Signal** : BEEPER_OUT (PA11 sur STM32WB55)
- **Sortie** : Mono, un seul canal
