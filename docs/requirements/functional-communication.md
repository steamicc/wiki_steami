# Exigences fonctionnelles - Communication (FCT-28 a FCT-34)

## Communication sans fil

| ID | Nom | Description |
|----|-----|-------------|
| FCT-28 | BLE | Le systeme doit pouvoir communiquer par Bluetooth BLE 4.2. |
| FCT-29 | BLE - Range | Le systeme necessite une portee physique BLE d'au moins 10 m. |
| FCT-30 | OpenThread | Le systeme doit pouvoir communiquer par OpenThread. |
| FCT-31 | ZigBee | Le systeme doit pouvoir communiquer par ZigBee. |
| FCT-32 | Antenna | Le systeme doit integrer une antenne interne. |

## Stockage

| ID | Nom | Description |
|----|-----|-------------|
| FCT-33 | QSPI Flash | Le systeme doit integrer un stockage flash Quad-SPI de 64 Mbit. |
| FCT-34 | QSPI Flash - USB Mass Storage | Le stockage flash interne doit etre accessible a travers USB Mass Storage. |
