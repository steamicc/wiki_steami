# Exigences fonctionnelles - Microcontroleur et stockage (FCT-35 a FCT-39)

| ID | Nom | Description |
|----|-----|-------------|
| FCT-35 | Microcontroller | Le systeme doit integrer un microcontroleur ARM Cortex-M STM32WB55 de STMicroelectronics. |
| FCT-36 | Microcontroller - RAM | Le systeme doit integrer un MCU avec suffisamment de RAM pour effectuer des operations graphiques de base (minimum pour un framebuffer 128x128). |
| FCT-37 | Microcontroller - Flash | Le systeme doit integrer un MCU avec au moins 512 Ko de memoire flash. |
| FCT-38 | Microcontroller - Speed | Le systeme doit integrer un MCU avec une vitesse de processeur d'au moins 64 MHz. |
| FCT-39 | Internal sensors - Bus | Tous les capteurs internes doivent etre connectes sur le premier bus I2C. |
