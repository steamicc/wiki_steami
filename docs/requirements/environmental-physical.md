# Exigences environnementales et physiques

## Exigences environnementales (ENV)

| ID | Nom | Description |
|----|-----|-------------|
| ENV-01 | Temperature | Le systeme doit pouvoir fonctionner dans la plage de -20 degC a 85 degC. |
| ENV-02 | Lifetime | Le systeme doit pouvoir fonctionner pendant 70 000 heures (environ 8 ans). |
| ENV-03 | ESD Protection | Les connecteurs d'entree et de sortie doivent etre equipes de protections ESD. |
| ENV-04 | User Interface Position | Le systeme doit integrer les elements d'affichage sur la face avant de la carte. |
| ENV-05 | Command Interface Position | Le systeme doit integrer l'interface de controle (boutons) sur la face avant de la carte. |

## Exigences physiques (PHY)

| ID | Nom | Description |
|----|-----|-------------|
| PHY-01 | Size | Le systeme doit etre contenu dans un volume inferieur a 100 mm de longueur par 60 mm de largeur. |
| PHY-02 | Weight | Le poids doit etre inferieur ou egal a 100 g. |
| PHY-03 | Battery integration | Le systeme peut integrer une batterie a l'arriere. |
