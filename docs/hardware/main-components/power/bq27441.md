# Jauge de batterie — BQ27441

La jauge BQ27441 mesure periodiquement l'energie restante dans la batterie, avec communication I2C et faible consommation.

## Caracteristiques

| Caracteristique        | Detail                                                   |
| ---------------------- | -------------------------------------------------------- |
| Modele                 | [BQ27441DRZR-G1A](https://www.ti.com/product/BQ27441-G1) |
| Fabricant              | Texas Instruments                                        |
| Interface              | I2C (bus I2C1 interne)                                   |
| Tension d'alimentation | 2.45V a 4.5V                                             |
| Technologie batterie   | LiCoO2, Li-Ion                                           |
| Boitier                | SON-12                                                   |
| Datasheet              | [BQ27441-G1](https://www.ti.com/product/BQ27441-G1)      |
