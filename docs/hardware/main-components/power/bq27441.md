# Jauge de batterie — BQ27441

La jauge BQ27441 mesure périodiquement l'énergie restante dans la batterie, avec communication I2C et faible consommation.

## Caractéristiques

| Caractéristique        | Détail                                                   |
| ---------------------- | -------------------------------------------------------- |
| Modèle                 | [BQ27441DRZR-G1A](https://www.ti.com/product/BQ27441-G1) |
| Fabricant              | Texas Instruments                                        |
| Interface              | I2C (bus I2C1 interne)                                   |
| Tension d'alimentation | 2.45V à 4.5V                                             |
| Technologie batterie   | LiCoO2, Li-Ion                                           |
| Boîtier                | SON-12                                                   |
| Datasheet              | [BQ27441-G1](https://www.ti.com/product/BQ27441-G1)      |
