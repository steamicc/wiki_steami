# Connecteur Jacdac

Interface série pour la connexion de modules Jacdac.

## Caractéristiques

| Caractéristique | Détail                              |
| --------------- | ----------------------------------- |
| Data            | JACDAC_DATA (UART2_TX/RX) → PB6/PB7 |
| Alimentation    | JACDAC_P : 5V                       |
