# Pads crocodile

Pads accessibles avec des pinces crocodile autour de la carte, permettant des connexions rapides sans soudure.

## Attribution des pads

| Pad                | Fonction                     |
| ------------------ | ---------------------------- |
| P0, P1, P2         | Entrées analogiques          |
| P24, P25, P28, P29 | GPIO via expandeur MCP23009  |
| 3V3                | Alimentation 3.3V            |
| GND                | Masse                        |
