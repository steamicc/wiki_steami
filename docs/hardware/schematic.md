## Reference design

Le reference design complet (Altium) est disponible sur GitHub :

- [steamicc/steami-reference-design](https://github.com/steamicc/steami-reference-design)

Il contient :
- Les fichiers schema Altium (`.SchDoc`)
- Le PCB (`.PcbDoc`)
- Les fichiers de fabrication (GERBER, BOM, Pick and Place)
- Le modele 3D ([STeaMi_3D.pdf](https://github.com/steamicc/steami-reference-design/blob/main/3D/STeaMi_3D.pdf))
- Le plan d'assemblage ([STeaMi-Assembly.PDF](https://github.com/steamicc/steami-reference-design/blob/main/Manufacturing/Assembly/STeaMi-Assembly.PDF))
