# Jauge de batterie

**Identifiant** : FP3

Cette fonction mesure periodiquement l'energie restante dans la batterie a l'aide de la jauge BQ27441 de Texas Instruments, qui inclut une communication I2C et une faible consommation.

## Caracteristiques

- **Composant** : Texas Instruments BQ27441DRZR-G1A
- **Interface** : I2C (bus interne I2C1)
- **Tension d'alimentation** : 2.45V a 4.5V
- **Technologie batterie supportee** : LiCoO2, Li-Ion
- **Boitier** : SON-12
