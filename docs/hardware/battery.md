# Batterie

**Identifiant** : FP1

La batterie assure le fonctionnement du systeme pendant une journee en mode standard et de 5 a 10 jours en mode basse consommation. Elle utilise la technologie LiPo pour une taille et un poids reduits.

## Caracteristiques

- **Technologie** : Lithium-Polymere (LiPo)
- **Tension** : 3.7V
- **Capacite** : 1600 mAh
- **Autonomie mode standard** : 1 jour
- **Autonomie mode basse consommation** : 5 a 10 jours
- **Integration** : La batterie peut etre integree a l'arriere de la carte
