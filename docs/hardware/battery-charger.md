# Chargeur de batterie

**Identifiant** : FP2

La batterie est rechargee via un port micro-USB avec un courant minimum de 500 mA. Le systeme peut continuer a fonctionner meme si la batterie est vide ou absente. Le design utilise le circuit de charge Texas Instruments BQ24075.

## Caracteristiques

- **Composant** : Texas Instruments BQ24075RGT
- **Courant de charge** : minimum 500 mA
- **Connecteur** : micro-USB
- **Mode de fonctionnement** : le systeme reste operationnel meme sans batterie (alimentation USB)
- **Gestion du chemin d'alimentation** : PowerPath integre
